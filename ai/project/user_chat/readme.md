1. pnpm i json-server
json-server是一个轻量级的工具，可以快速创建REST API服务，常用于开发和测试环境中模拟后端数据服务。

2. npm run dev
npm run dev 执行的是在 package.json 的 scripts 字段中定义的 dev 脚本
启动 json-server 服务，监听端口 3001，默认提供 /users 路由，数据存储在 users.json 文件中。

- 以.json 结尾和 以.sql 结尾的文件，都可以作为数据文件来处理
- package.json文件中的 script 是后端脚本
