# 绘本岛 - 亲子阅读与活动平台

## 🌟 项目介绍

绘本岛是一个面向中国亲子家庭的"绘本阅读 + 亲子活动"平台，致力于让亲子时光更温暖。平台提供海量优质绘本阅读和丰富的线下亲子活动，帮助家长培养孩子的阅读兴趣，增进亲子关系。

## 🎯 核心功能

### 📚 绘本阅读
- **绘本馆**: 分类展示海量绘本，支持按年龄段、主题筛选
- **智能搜索**: 支持书名、作者关键词搜索
- **个性化推荐**: 基于孩子年龄和兴趣推荐合适绘本
- **阅读记录**: 记录阅读历史，支持打卡和写读后感

### 🎪 亲子活动
- **活动发布**: 线下亲子活动（故事会、手工课、户外探索等）
- **活动筛选**: 按地区、类型、时间筛选活动
- **在线报名**: 便捷的活动报名和支付流程
- **活动提醒**: 活动开始前自动提醒

### 👨‍👩‍👧‍👦 用户中心
- **多孩档案**: 支持管理多个孩子的信息
- **成长记录**: 记录孩子的阅读成长足迹
- **成就徽章**: 激励孩子持续阅读
- **订单管理**: 查看绘本购买和活动报名订单

## 🎨 设计特色

### 视觉风格
- **主色调**: 米白(#FEF7F0)、淡黄(#FFF8DC)、天蓝(#87CEEB)
- **设计风格**: 温馨、清新、富有童趣
- **插画风格**: 柔和的水彩风格，适合亲子主题
- **图标设计**: 统一使用lucide-react图标库

### 交互体验
- **移动端优先**: 完全适配手机端浏览
- **响应式设计**: 支持平板和桌面端
- **交互动效**: 细腻自然的过渡动画
- **操作简化**: 专为家长设计的简洁操作流程

## 🛠 技术架构

### 前端技术栈
- **React 18**: 现代化的用户界面库
- **TypeScript**: 类型安全的JavaScript超集
- **Vite**: 快速的构建工具和开发服务器
- **Tailwind CSS**: 实用优先的CSS框架
- **React Router**: 单页应用路由管理
- **Zustand**: 轻量级状态管理

### 项目结构
```
src/
├── components/          # 可复用组件
├── pages/              # 页面组件
├── hooks/               # 自定义React Hooks
├── utils/               # 工具函数
├── stores/              # 状态管理
└── types/               # TypeScript类型定义
```

## 🔧 开发指南

### 代码规范
- 使用TypeScript确保类型安全
- 组件文件使用.tsx扩展名
- 遵循React函数组件和Hooks最佳实践
- 使用Tailwind CSS进行样式设计

### 组件设计原则
- 保持组件小巧专注（< 300行）
- 使用组合而非继承
- 提取可复用逻辑到自定义Hooks
- 保持组件纯净和可测试性

### 状态管理
- 使用Zustand进行全局状态管理
- 组件内部状态使用useState
- 复杂状态逻辑使用useReducer
