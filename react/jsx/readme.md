# JSX 语法

1. JSX 是一种 JavaScript 的语法扩展，用于描述用户界面。
2. 它看起来很像 HTML，但实际上是 JavaScript 的一种特殊语法，用于创建 React 元素。
3. JSX 可以在 React 组件中使用，用于定义组件的结构和行为。

- JSX 
  react 的一个亮点
  react 的模板语法，在 JS 里写 HTML
  jsx: xml in js, html 是一种特定的 xml

- vue/react 现代前端框架的特性

响应式 reactive

数据绑定 data binding

组件化 component

React 更激进，入门门槛比较高
