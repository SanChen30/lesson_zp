# props

1. state，状态是变化的数据，自有的

2. props，传递的数据，父组件传递过来的

## 组件 components

1. 拼积木一样，把页面拼出来
2. 开发任务的最小单元
3. 组件可以封装，components 目录下放子组件；pages 目录下放路由，页面级别组件
4. 复用性，组件类
5. 协作
6. 组件可以嵌套
   1. 父子组件
   2. App.jsx，老板
   3. Greeting.jsx，员工
7. 组件之间就会有数据的传递


## 组件通信

1. 父组件一般负责持有数据，useState 自有的，私有的
2. 子组件可以拿到父组件传递的props，props 是只读的，不能在子组件中修改
   function 的 js 意义，通过参数传递数据，即 props 是一个参数对象，可以在自定义组件中书写属性，属性即是 props 的 key
   组件闭合开关的中间书写的内容，就是 props.children
