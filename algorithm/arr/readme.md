## 线性数据结构
1) 数组: 常用、好用、连续内存
2) 栈: 先进后出 FILO
3) 队列: 先进先出 FIFO
4) 链表: 离散、不连续内存

## 非线性数据结构
1) 树（二叉树）
2) 图: 顶点和边的集合，顶点之间可以通过边连接


## 数组，开箱即用的数据结构

1. 数组的创建

- 知道数组的每一项
  const arr = [1, 2, 3, 4, 5, 6];
- 不知道数组内部元素情况
  const arr = new Array(10);
  要多了 浪费
  要少了 JS 支持动态扩容，二次申请空间，开销比较大
    ```js
    const arr2 = []; // 初始容量很小
    for (let i = 0; i < 1000; i++) {
        arr2.push(i); // 可能触发多次扩容操作
    }
    // 扩容操作涉及：分配新内存 + 复制元素 + 回收旧内存
    ```

考虑动态性的时候，数组是比链表差的
数据量小的情况下，数组是更优秀的，因为链表节点的实例化需要用到多个指针
