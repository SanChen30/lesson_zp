# 反转字符串 abc -> cba

1. 面试官的内心

1) api 熟练度

2) 代码的逻辑能力

- 很多种解法

    1. api
    2. for
    3. 递归：函数自己调用自己，整个字符串的反转，拆解为，每次反转一个字符，然后递归调用，直到字符串为空，爆栈（风险），内存开销比较大
    4. for of
    5. ...展开运算符：将字符串转换为数组，`[...str]`等价于 `str.split('')`
    6. 数组的 reduce 方法：从左到右遍历数组，每次遍历将当前元素和上一次遍历的结果合并，返回合并后的结果
