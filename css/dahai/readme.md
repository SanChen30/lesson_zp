# CSS 大海

1. 何为CSS？

一个属性与值的键值对为一个声明

声明块：将多个声明组成一个声明块

声明块如果需要作用到对应的HTML元素，需要使用选择器

cssRules 组成了样式表

- 层叠？

- 外边距合并

- 小数单位的px 怎么处理

- inline 元素在有些情况下不支持transform属性，也不支持改变宽高、上下margin等属性

- CSS选择器优先级按个十百千位计算

- 每个选择器的优先级是一个四位数，分别对应!important、内联样式、ID选择器、类选择器、标签选择器

- !important 优先级最高，但是不建议使用，因为会导致代码维护困难
