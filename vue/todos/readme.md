# Todos 任务清单 vue 开发思路

- 响应式数据驱动的界面开发
- v-if/v-for/v-model
- composition API
  - @ 事件绑定
  - :class 动态类绑定
  - computed 计算属性
  - vue 开发方式 简单、高效、好上手

1. 传统做法

先找到目标元素，然后对其做修改，也就是 DOM 操作

2. vue 做法

不在需要思考页面的元素怎么操作，而是要思考数据是怎么变化的

1) v-for 循环输出

v- 指令 

2) computed 计算属性

永远不要在模板中写复杂或重复的逻辑表达式，把它移到 computed（或方法）中。

性能更好，只有在依赖的数据变化时才会重新计算
