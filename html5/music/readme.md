## 代码敲击乐
- html5 模拟钢琴 **模块化**开发
  - html 负责结构 键盘布局
    - !+tab 快速生成html结构
    - html 文档（Document）
    - html第5个版本 <!DOCTYPE html> html5文档头声明
    - <></> 标签对
    - **第一步**写结构
  - css 负责样式 键盘外观
    - **第二步**写样式
  - js 负责交互 键盘音效

## HTML 标签分为两类
- 块级元素（block）
  - 独占一行
  - 可以设置宽度、高度、内边距、外边距
  - 宽度默认整行
  - 高度如果没有设定由内容撑开
- 行内元素（inline）
  - 不独占一行
  - 不能设置宽度、高度、内边距、外边距
