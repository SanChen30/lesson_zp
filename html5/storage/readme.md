## 关于 box-shadow

box-shadow: [inset] 水平偏移 垂直偏移 模糊半径 扩散半径 颜色;

记住水平偏移、垂直偏移必填，其余可选。

对比其他方式
方式	            是否跟随border-radius	        是否占布局空间	        能否设透明度
border	            ✅ 是	                          ✅ 是	            ❌（除非用 rgba，但仍是边框）
outline	            ❌ 否（通常是直角）	                ❌ 否	           ✅
box-shadow	        ✅ 是	                          ❌ 否	             ✅

所以：如果想要一个带圆角、不占空间、可透明的“边框”，就用这种 box-shadow！


## outline 轮廓

有点像border，但是不会占用布局空间，也不会影响布局。

## overflow 溢出

可以管住子元素的溢出，防止子元素超出父元素的范围。

## flex 布局

创建一个新的格式化上下文，内部元素的布局将基于这个上下文进行计算。
