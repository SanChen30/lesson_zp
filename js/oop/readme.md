# OOP 面向对象编程

Object Oriented Programming 面向对象编程

## 封装

JS 是一种**基于**对象（Object-based）的语言，你几乎遇到的所有东西都是对象（连简单数据类型都有包装类）。

它又不是一个真正的面向对象（OOP），早期连 class 关键字都没有，哪怕 es6 有了 class，但仍然是原型式的面向对象。

1. 如果没有 Constructor, 没有 class, 怎么做面向对象

对象字面量

## 生成实例对象的原始模式

对象字面量虽好，但是我们还是要封装一下，生成实例对象的过程，封装成一个函数，这个函数就是构造函数。

1. 实例生成的过程

函数以 new 的方式调用

函数内部创建一个空对象，this 指向这个空对象

执行构造函数里面的代码，完成空对象的实例化，空对象就有类模板的属性了

默认返回这个实例对象

cat2.constructor === cat3.constructor 表现了实例对象之间的关系

## Prototype 模式

把不变的属性和公用的方法，都放到原型对象上

所以 JS 是原型式的面向对象编程。

## 继承

extends 关键字
