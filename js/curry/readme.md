# 柯理化函数

- 柯理化函数的定义：柯理化函数是指将一个多参数函数转换为一系列单参数函数的技术。

- 函数需要的参数，需要一个一个的传递，比如大模型返回，异步执行等等，不能一次性拿到所有参数。

- 严格柯理化函数：柯理化函数的参数，只能一个一个的传递，不能一次传递多个参数。

- 非严格柯理化函数：柯理化函数的参数，既可以一个一个的传递，也可以一次传递多个参数。

- 本质是闭包，闭包的作用是记住函数的参数，形成闭包链。 
  每一层函数都可以接收自己的参数
  当参数数量足够时，返回结果
  当参数数量不足时，返回新的函数，等待下一个参数的传递
- 什么时候够？
  fn.length 返回函数参数数量
- 闭包 + 递归 实现柯理化函数
  - 闭包中的自由变量一直在，不会销毁，用于不断收集参数（递归中重复的事情）
  - 退出条件，参数数量等于函数参数数量，执行函数
